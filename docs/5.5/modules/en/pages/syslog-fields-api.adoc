== Security Architecture & Gateway Authentication Definitions
All transactions dispatched to NeuVector endpoints must execute valid cryptographic parameters inside incoming HTTP header vectors. Missing tokens fallback directly into a `HTTP 401 Unauthorized` block state configuration.

[cols="2a,1a,2a,1a", options="header", frame="all", grid="all"]
|===
| Security System Token | Delivery Context Layer | Destination HTTP Header Field | Cryptographic Type Baseline
| *ApiKeyAuth* | `header` | `X-Auth-Apikey` | `apiKey`
| *TokenAuth* | `header` | `X-Auth-Token` | `apiKey`
|===
