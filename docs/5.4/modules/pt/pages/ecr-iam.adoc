== AWS ECR - Funções IAM

Quando os contêineres {product-name} são implantados na AWS, e uma instância EC2 é atribuída a uma função de "`EC2 Container Registry`" Acesso de Leitura, o registro AWS ECR pode ser digitalizado sem uma Chave de Acesso e Chave Secreta.

Aqui está como criar uma função AWS e atribuí-la ao nó.

=== Selecione a Instância

Observe que a função IAM está em branco ou não inclui a função ECR

image:ecr1.png[awsrole]

=== Anexar uma Função

Selecione Ações -> Configurações da Instância -> Anexar/Substituir Função IAM

image:ecr2.png[awsrole]

Se você não criou anteriormente a função ECR, clique em Criar Nova Função IAM. Digite o nome da função.

image:ecr3.png[attachrole]
