== Detección de Amenazas de Red

{product-name} la inspección profunda de paquetes puede ser utilizada para inspeccionar los paquetes de red y la carga útil en busca de ataques como los que se encuentran en el OWASP Top 10 y aquellos comúnmente utilizados en los firewall de aplicaciones web (WAFs).

=== Detección de Amenazas Integrada

{product-name} también tiene detección integrada de otras amenazas de red como ataques de inyección SQL, DDoS (por ejemplo, Ping de Muerte) y ataques de tunelización. Para ataques de inyección SQL, {product-name} inspecciona la conexión de red (protocolo SQL) entre el front end y el pod de la base de datos SQL, reduciendo falsos positivos y aumentando la precisión.
