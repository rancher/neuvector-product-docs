== Installation et configuration

=== Obtenir l'appli

==== GitHub

Téléchargez le dernier fichier tarball de l'appli (`neuvector_app.tar.gz`) depuis le https://github.com/neuvector/neuvector-splunk-app[dépôt neuvector/neuvector-splunk-app].

==== Splunkbase

Téléchargez le dernier fichier tarball de l'appli depuis https://splunkbase.splunk.com/apps[Splunkbase].

==== Navigateur d'applis Splunk

Dans l'interface Splunk, cliquez sur le menu déroulant Applications, cliquez sur "Trouver d'autres applications", puis recherchez l'appli Splunk {product-name}.

=== Installation et configuration

Installez l'appli en téléchargeant le fichier tarball ou en suivant les instructions de Splunkbase.

==== Configurez syslog dans {product-name} la console

Allez dans Paramètres -> Configuration -> Syslog

[loweralpha]
. définissez la valeur du serveur comme l'adresse IP sur laquelle Splunk fonctionne
. choisissez TCP comme protocole
. définissez le numéro de port sur 10514
. choisissez le niveau d'information
. cliquez sur SOUMETTRE pour enregistrer le paramètre.

image:syslog-config.png[syslog]

Vous pouvez configurer plusieurs clusters pour envoyer des syslogs à votre instance Splunk et votre instance Splunk recevra ces syslogs en temps réel.
